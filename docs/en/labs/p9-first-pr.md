# P9 First PR

## Goal

Enter open source contribution through small-scale, reproducible, and verifiable work.

## Recommended Entry Points

- Fix documentation or example commands.
- Add minimal reproduction.
- Add benchmark records or tests.
- Fix small-scope bugs.

## Pass Criteria

Before submitting, you can explain: what the problem is, how to reproduce it, what you changed, how to verify it, and where the risks are.
